## Requisitos

- [Bun](https://bun.sh) >= 1.2

## Desarrollo

```bash
bun install
bun dev
```

## Producción

```bash
bun run build
bun start
```

## Tecnologías

- **Framework:** Next.js 16 (App Router)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS v4
- **Runtime:** Bun
